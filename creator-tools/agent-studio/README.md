## Eternalai Agent Studio

### Installing

```bash
$ npm install eternal-ai-agent-studio
```

### Exports
```js
// ES6
import {Studio, StudioProps} from 'eternal-ai-agent-studio';
```

### Usage
```js
import {Studio, StudioProps} from 'eternal-ai-agent-studio';

function App() {
  return (
    <Studio
      categories={[]}
      data={[]}
    >
    </Studio>
  );
}
ReactDOM.render(<App/>, document.body);
```
